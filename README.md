
# 🥗 NutriLens (Public Overview)

**NutriLens** is an iOS nutrition tracking concept app built with **SwiftUI**. It focuses on clean UI, smooth interactions, and modular architecture.

## 📱 Features (High-Level)

* AI-inspired **food scanning** concept
* **Daily meal tracking**
* **Progress & statistics** overview
* Minimal **rewards & achievements** idea
* Modern **glassmorphic UI style**

*(Note: This public version contains only a conceptual/preview-level overview of the app.)*

## 🏗️ Architecture

NutriLens uses a modular structure inspired by **Clean Architecture**:

* **Presentation** – SwiftUI Views & ViewModels
* **Domain** – Business rules
* **Data** – Repository & service implementations

## 🛠️ Tech Stack

* **Swift 6**
* **SwiftUI** (iOS 26+)
* **MVVM + Modular Architecture**
* **Async/Await**
* **NavigationStack**

## 🚀 Getting Started

```bash
git clone https://github.com/prajapati-nikunj/NutriLens
open NutriLens.xcodeproj
```

* Requires **Xcode 16 / iOS 26+**
